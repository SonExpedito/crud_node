# **crud_node - *NIKE***

Repositório referente ao primeiro projeto CRUD com Node.js, feito com estilização em **TailwindCSS** e utilizando as seguintes bibliotecas:

- Express  
- Express-Handlebars  
- MySQL2  
- Body-Parser  
- Sequelize
